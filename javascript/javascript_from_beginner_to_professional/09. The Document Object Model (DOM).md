## El BOM (Modelo de Objetos del Navegador)
- El BOM, también llamado el objeto de **ventana del navegador**, es el elemento que hace posible que tu código JavaScript se comunique con el navegador.
- El objeto `window` contiene todas las propiedades requeridas para representar la ventana del navegador, como, por ejemplo, el tamaño de la ventana y el historial de páginas web visitadas anteriormente.
- El objeto `window` tiene variables y funciones globales, y todas se pueden ver cuando exploramos el objeto `window`.

En la consola del navegador, navega a la pestaña **Console** (Consola). Escribe el siguiente comando y presiona Enter para obtener información sobre el objeto `window`:
```javascript
console.dir(window);
```

Este comando producirá una vista como la siguiente:
![[Pasted image 20250929112747.png]]
El método `console.dir()` muestra una lista de todas las propiedades del objeto especificado.

El BOM contiene muchos otros objetos. Por ejemplo, podemos obtener la longitud del historial (en mi navegador) accediendo al objeto `history` de la `window` y luego a la propiedad `length` del objeto `history`:
```javascript
window.history.length;
```

### Objeto `history` de la ventana
El objeto `window` del navegador también contiene un objeto `history`. Se puede escribir sin el prefijo `window` porque se ha hecho globalmente disponible (`console.dir(window.history) == console.dir(history)`).

Este objeto lo puedes usar para volver a una página anterior, porque tiene una función incorporada llamada `go`:
```javascript
window.history.go(-1);
```

### Objeto `navigator` de la ventana
En el objeto `window` hay una propiedad `navigator`. Esta propiedad contiene información sobre el navegador que estamos usando, como qué navegador es y qué versión usamos y en qué sistema operativo se ejecuta el navegador:
```javascript
console.dir(window.navigator);
// or
console.dir(navigator);
```

### Objeto `location` de la ventana
Contiene la URL de la página web actual. Si anulas (partes de) esa propiedad, obligas al navegador a ir a una nueva página.
```javascript
console.dir(window.location)
// or
console.dir(location)
```

## El DOM (Modelo de Objetos del Documento)
El DOM es una forma de mostrar la estructura de un documento HTML como un árbol lógico:
![[Pasted image 20250929220456.png]]
### Propiedades adicionales del DOM
Podemos inspeccionar el DOM de manera similar a como lo hicimos con los otros:
```javascript
console.dir(document);
```
Veremos el `objeto`, que representa el DOM. El DOM contiene todos los elementos HTML de una página web.

### Seleccionar elementos de la página
El objeto `document` contiene muchas propiedades y métodos. Para trabajar con elementos en la página, primero tendrás que encontrarlos.

Para seleccionar elementos de la página para usar dentro de tu código JavaScript y para manipular elementos, puedes usar los métodos `querySelector()` o `querySelectorAll()`. Ambos se pueden usar para seleccionar elementos de la página por nombre de etiqueta, ID o clase:
- El método `document.querySelector()` devolverá el primer elemento dentro del documento que coincida con los selectores especificados. Si no se encuentran elementos de página coincidentes, se devuelve el resultado `null`.
- Para devolver múltiples elementos coincidentes, puede usar el método `document.querySelectorAll()`.

El método `querySelectorAll()` devolverá una **NodeList** estática, que representa una lista de los elementos del documento que coinciden con el grupo especificado de selectores.

Demostraremos el uso de `querySelector()` y `querySelectorAll()` con el siguiente fragmento HTML:
```html
<!doctype html>
<html>
<head>
    <title>Probador JS</title>
</head>
<body>
    <h1 class="output">Hello World</h1>
    <div class="output">Test</div>
    <script>
        const ele1 = document.querySelector("h1");
        console.dir(ele1);

        const eles = document.querySelectorAll(".output");
        console.log(eles);
    </script>
</body>
</html>
```
Puedes comenzar a usar las características dinámicas del DOM: puedes **manipular** los elementos usando JavaScript.
