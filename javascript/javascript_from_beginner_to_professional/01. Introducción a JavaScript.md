## ¿Cómo entiende el navegador a JavaScript?
JavaScript es un lenguaje interpretado, el ordenador lo entiende mientras lo ejecuta. Adicionalmente, la especificación o estandarización del lenguaje JavaScript se lo conoce como ECMAScript (abreviado ES), los navegadores utilizan esta especificación.
## Uso de la consola del navegador
En casi todos los navegadores web, pulsando F12 o clic con el botón derecho, seleccionando "Inspeccionar"  y luego buscamos la pestaña "console" (consola) podemos introducir comandos de JavaScript y ejecutarlos:
```javascript
console.log("¡Hola mundo!");
```
## ¿Cómo añadir JavaScript a una página web?
Hay dos formas de vincular JavaScript a una página web: 
### Directamente en HTML
```html
<html>
	<script type="text/javascript">
		alert("Hi there!");
	</script>
</html>
```
### Vincular un archivo externo
Archivo `ch1_alert.js`:
```javascript
alert("¡Hola desde otro archivo!");
```

Vinculamos al archivo HTML:
```html
<html>
	<script type="text/javascript" src="ch1_alert.js"></script>
</html>
```
## Escribir código JavaScript (recomendaciones)
### Puntos y comas
Después de cada instrucción, debes poner un punto y coma.  JavaScript igualmente ejecutará si no pones punto y coma, pero es recomendable añadir después de cada línea de código, excepto en bloques de código como if o un bucle.
### Comentarios
Hay dos formas de escribir comentarios:
```javascript
// Soy un comentario de una sola línea

/*
Soy un comentario de varías líneas.
Todo lo que se encuentre entre la barra y el asterisco
y la barra no se ejecutará.
*/
```
### Prompt (Solicitud)
Este comando pedirá al usuario que ingrese una palabra o número:
```javascript
prompt("¡Hola!, ¿Cómo estás?"); // Bien
```
El valor que se introduzca se devolverá al script y se podrá utilizar en el código.
## Números aleatorios
Genera un número decimal entre 0 y 1:
```javascript
Math.random();
```

Si queremos un número 0 y 100:
```javascript
Math.random() * 100;
```

Si no queremos un número decimal, podemos usar la función `Math.floor` que redondea el resultado al entero más cercano:
```javascript
Math.floor(Math.random() * 100);
```
