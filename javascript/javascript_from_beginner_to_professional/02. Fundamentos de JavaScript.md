## Declaración variables
Antes de crear una variable, **se declara**. Y para ello puedes poner *let, var o const*:
### let, var y const
- Tanto *let* como *var* se utilizan para variables que por alguna razón su valor tendríamos que cambiarlo en alguna parte del programa. **Su valor no es fija, puede cambiar.
-  `var` tiene alcance global y `let` alcance de bloque.
- Por otro lado, *const* se utiliza para variables a las que se valor no cambiará nunca. Por ejemplo el valor `pi` 
### Nombrar variables
- Las variables deben comenzar con una letra minúscula.
- No pueden contener espacios, pero sí pueden usar guiones bajos.
## Tipos de datos primitivos
JavaScript en un lenguaje de tipado flexible, es decir, determina el tipo en función del valor. **No es necesario declarar el tipo de dato.** JavaScript tiene siete tipos de datos primitivos:
### String
Se utiliza para almacena un valor de texto y hay varias maneras de declararlo:
	- Con comillas dobles.
	- Comillas simples.
	- Comillas invertidas.

En una cadena con comillas invertidas se pueden insertar variables en el mensaje:
```javascript
let language = "JavaScript";
let message = `Let's learn ${language}`;
console.log(message); // Let's learn JavaScript
```
#### Caracteres de escape `\`
El carácter de escape indica al intérprete que trate el siguiente carácter de forma especial (no con su significado habitual):
```javascript
let str = "Hello, what's your name? Is it \"Mike\"?";
console.log(str); // Hello, what's your name? Is it "Mike"?
let str2 = 'Hello, what\'s your name? Is it "Mike"?';
console.log(str2);//Hello, what's your name? Is it "Mike"?
```

El carácter de escape también sirve para crear saltos de línea (`\n`) o para incluir una barra invertida (`\\`):
```javascript
let str3 = "New \nline.";
let str4 = "I'm containing a backslash: \\!";
console.log(str3);
/*
New
line.
*/
console.log(str4); // I'm containing a backslash: \!
```
### Number
Almacena números (enteros, decimales, con signo, sin signo, etc.). También puedes representar números en notación exponencial, octal, hexadecimal y binaria:
```javascript
let intNr = 1;
let decNr = 1.5;
let expNr = 1.4e15;
let octNr = 0o10;
let hexNr = 0x3E8;
let binNr = 0b101;
```
### BigInt
Si necesitas manejar números muy grandes (o muy pequeños en magnitud entera), puedes usar BigInt; se reconoce por el sufijo n:
```javascript
let bigNr = 90071992547409920n;
```
Los BigInt solo pueden mezclarse en operaciones con otros BigInt.
### Boolean
Puede contener dos valores: **true** o **false**.
```javascript
let bool1 = false;
let bool2 = true;
```
### Symbol
Se usa para crear identificadores únicos; aun cuando dos Symbols tengan la misma descripción, son distintos:
```javascript
let str1 = "JavaScript is fun!";
let str2 = "JavaScript is fun!";
console.log("These two string are the same: ", str1 === str2); // true

let sym1 = Symbol("JavaScript is fun!");
let sym2 = Symbol("JavaScript is fun!");
console.log("These two Symbols are the same:", sym1 === sym2); // false
```
### Undefined
Tipo para variables que no tienen un valor asignado:
```javascript
let unassigned;
console.log(unassigned); // undefined
```
### null
Valor especial que representa ausencia intencional de un valor. Nota: typeof null devuelve "object" por legado histórico.
```javascript
let terribleThingToDo = undefined;
let lastName;
console.log("Same undefined:", lastName === terribleThingToDo); // true
let betterOption = null;
console.log("Same null:", lastName === betterOption); // false
```
## Analizar y convertir tipos de datos
JavaScript incluye herramientas para identificar y transformar tipos primitivos.
### Determinar el tipo de una variable
El operador `typeof` devuelve el tipo de una expresión:
```javascript
let str = "Hello";
let nr = 7;
let bigNr = 12345678901234n;
let bool = true;
let sym = Symbol("unique");
let undef = undefined;
let unknown = null;

console.log("str", typeof str);         // str      "string"
console.log("nr", typeof nr);           // nr       "number"
console.log("bigNr", typeof bigNr);     // bigNr    "bigint"
console.log("bool", typeof bool);       // bool     "boolean"
console.log("sym", typeof sym);         // sym      "symbol"
console.log("undef", typeof undef);     // undef    "undefined"
console.log("unknown", typeof unknown); // unknown  "object"  // legado histórico
```
`typeof` es un operador, no una función, por eso se usa sin paréntesis.
### Convertir tipos de datos
JavaScript hace coerción automática en muchos casos, pero también puedes convertir explícitamente con:
	- `String(value)`: convierte cualquier valor a cadena.
	- `Number(value)`: intenta convertir a número; si no es posible devuelve NaN.
	- `Boolean(value)`: convierte a booleano. Los valores que se consideran falsy son: false, 0, -0, 0n, "", null, undefined y NaN; todo lo demás es truthy.
```javascript
String(123);        // "123"
Number("12.34");    // 12.34
Number("foo");      // NaN
Boolean(0);         // false
Boolean("hello");   // true
```
## Operadores
### Operadores aritméticos
#### Suma
Si ambos operandos son números, realiza la suma; si alguno es cadena, realiza concatenación.
#### Resta
Resta numérica (si alguno no es número, se intenta convertir; si no es posible, resulta NaN).
#### Multiplicación
Multiplica valores numéricos usando `*`. A diferencia de Python, multiplicar número por cadena no produce repetición: dará NaN si la conversión falla.
#### División
Divide con `/`.
#### Exponenciación
Operador `**` (p. ej. `x ** y`).
#### Módulo
Resto de la división, operador `%`.
#### Operadores unarios: incremento y decremento
- `x++` equivale a `x = x + 1` (pero devuelve el valor anterior).  
- `x--` equivale a `x = x - 1` (devuelve el valor anterior).
#### Prefijo y sufijo
- Sufijo (`x++`, `x--`): devuelve el valor antes de cambiarlo.
- Prefijo (`++x`, `--x`): incrementa/decrementa y devuelve el nuevo valor.
```javascript
let nr1 = 2;
console.log(nr1++); // 2
console.log(nr1);   // 3

let nr2 = 2;
console.log(++nr2); // 3
console.log(nr2);   // 3
```
#### Precedencia
| Grupo                          | Símbolos             | Ejemplo                   |
| ------------------------------ | -------------------- | ------------------------- |
| Agrupación                     | `(...)`              | `(x + y)`                 |
| Exponenciación                 | `**`                 | `x ** y`                  |
| Prefijo (inc/dec, unary)       | `++`, `--`, `+`, `-` | `++x`, `-x`               |
| Multiplicación/División/Módulo | `*`, `/`, `%`        | `x * y`, `x / y`, `x % y` |
| Suma y resta                   | `+`, `-`             | `x + y`, `x - y`          
Cuando hay operadores del mismo nivel de precedencia, se opera de izquierda a derecha.
### Operadores de asignación
El operador básico de asignación es `=`.  También podemos usar los operadores compuestos que combinan asignación con una operación aritmética:
```javascript
x += 5 // equivale a x = x + 5
x -= 5 // equivale a x = x - 5
x *= 3 // equivale a x = x * 3
x /= 2 // equivale a x = x / 2
x %= 4 // equivale a x = x % 4
x **= 3 // equivale a x = x ** 3
```
Estos operadores existen para todos los operadores aritméticos habituales.
### Operadores de comparación
Las comparaciones devuelven siempre un valor booleano.
#### Igualdad
- `==` — igualdad por valor (realiza coerción de tipos): devuelve true si los valores son equivalentes tras conversión.
- `===` — igualdad estricta (valor y tipo): devuelve true solo si valor y tipo coinciden.

Se recomienda usar `===` salvo que realmente quieras la coerción de `==`.
#### Desigualdad
- `!=` — distinto por valor (con coerción).
- `!==` — distinto estricto (valor o tipo distintos).
#### Mayor/menor
Operadores habituales: `>`, `<`, `>=`, `<=`.
```javascript
let nb1 = 5;
let nb2 = 6;

console.log(nb1 > nb2);  // false
console.log(nb1 < nb2);  // true
console.log(nb1 >= nb2); // false
console.log(nb1 <= nb2); // true
```
### Operadores lógicos
Devuelven booleanos y permiten combinar condiciones:

- AND lógico: `&&` — true si ambas expresiones son truthy.
```javascript
let x = 1, y = 2, z = 3;
console.log(x < y && y < z); // true
```

- OR lógico: `||` — true si al menos una expresión es truthy.
```javascript
console.log(x > y || y < z); // true
```

- NOT lógico: `!` — invierte el valor booleano.
```javascript
let flag = false;
console.log(!flag); // true

let a = 1, b = 2;
console.log(!(a < b)); // false
```
---
- Los operadores de comparación y lógicos trabajan con valores truthy/falsy; valores falsy: `false`, `0`, `-0`, `0n`, `""`, `null`, `undefined`, `NaN`.
- Las operaciones lógicas usan evaluación cortocircuito: en `A && B`, si A es falsy devuelve A sin evaluar B; en `A || B`, si A es truthy devuelve A sin evaluar B.
