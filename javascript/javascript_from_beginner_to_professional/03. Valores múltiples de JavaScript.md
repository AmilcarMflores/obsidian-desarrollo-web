## Arrays y sus propiedades
Los arrays son listas de valores, estos valores pueden ser de cualquier tipo de dato, e incluso un array puede contener diferentes tipos. 
### Creando array
```javascript
let arr = ["hi there", 5, true];
console.log(typeof arr[0]); // string
console.log(typeof arr[1]); // number
console.log(typeof arr[2]); // boolean
```
### Acceder a los elementos
```javascript
cars = ["Toyota", "Renault", "Volkswagen"];
console.log(cars[0]); // Toyota
console.log(cars[1]); // Renault
console.log(cars[2]); // Volkswagen
```
### Sobrescribiendo elementos
```javascript
cars = ["Toyota", "Renault", "Volkswagen"];
cars[0] = "Tesla";
console.log(cars); // [ 'Tesla', 'Renault', 'Volkswagen' ]
```
### Propiedad length integrada
```javascript
colors = ["black", "orange", "pink"];
booleans = [true, false, false, true];
emptyArray = [];

console.log("Length of colors:", colors.length); // 3
console.log("Length of booleans:", booleans.length); // 4
console.log("Length of emptyArray:", emptyArray.length); // 0
```
## Métodos de array
### Agregar y reemplazar elementos
Podemos agregar elementos con el método `push()`:
```javascript
favoriteFruits = ["apple", "banana", "cherry"];
favoriteFruits.push("tangerine");
console.log(favoriteFruits); // ["apple", "banana", "cherry", "tangerine"]
```

Si quieres añadir elementos en un índice determinado usamos `splice()`:
```javascript
let arrOfShapes = ["circle", "triangle", "rectangle", "pentagon"];
arrOfShapes.splice(2, 0, "square", "trapezoid");
console.log(arrOfShapes); // ['circle', 'triangle', 'square', 'trapezoid', 'rectangle', 'pentagon']
```
El segundo parámetro del método splice indica cuántos elementos se deben eliminar a partir del índice especificado.

También se puede agregar otro array al array con `concat()`:
```javascript
let arr5 = [1, 2, 3];
let arr6 = [4, 5, 6];
let arr7 = arr5.concat(arr6);
console.log(arr7); // [1, 2, 3, 4, 5, 6]

let arr8 = arr7.concat(7, 8, 9);
console.log(arr8); // [1, 2, 3, 4, 5, 6, 7, 8, 9]
```
### Eliminar elementos
Hay varias maneras:

- Si queremos eliminar el último elemento de la lista usaremos `pop()`:
```javascript
let arr = [1, 2, 3, 4, 5];
arr.pop();
console.log(arr); // Output: [1, 2, 3, 4]
```

- Podemos eliminar el primer elemento con `shift()`:
```javascript
let arr = [1, 2, 3, 4, 5];
arr.shift();
console.log(arr); // Output: [2, 3, 4, 5]
```

- También podemos eliminar con el método que ya vimos, `splice()`:
```javascript
let arr = [1, 2, 3, 4, 5];
arr.splice(1, 3);
console.log(arr); // Output: [1, 5]
```
Especificamos desde el índice que empezará a eliminar y el número de elementos que eliminará.

Si no queremos modificar el tamaño del array, podemos usar el operador `delete`:
```javascript
let arr = [1, 2, 3, 4, 5];
delete arr[2];
console.log(arr); // [1, 2, <1 empty item>, 4, 5]
console.log(arr.length); // 5
console.log(typeof arr[2]); // "undefined"
```
### Encontrar elementos

- Para comprobar si un valor está en una matriz, usaremos la función `find()`:
```javascript
let arr = [2, 4, 6, 8];
let findValue = arr.find(function(e) {return e === 6});
let findValue2 = arr.find(e => e === 10);
console.log(findValue, findValue2); // 6, undefined
```
Se ejecutará hasta encontrar una coincidencia y nos devolverá el elemento, si no la encuentra nos dará undefined.

- Si queremos saber la posición del elemento buscado, usaremos `indexOf()`:
```javascript
let arr = [2, 4, 6, 8];
let findIndex = arr.indexOf(6);
let findIndex2 = arr.indexOf(10);
console.log(findIndex, findIndex2); // 2 -1
```
Si un valor aparece más de una vez, devolverá la primera ocurrencia, si no encuentra, devolverá -1. 

Pero si quiere encontrar la próxima aparición, puede agregar un segundo argumento a `indexOf()`, especificando desde qué posición debe comenzar la búsqueda:
```javascript
let arr = [2, 4, 6, 8];
let findIndex3 = arr.indexOf(6, 3);
console.log(findIndex3); // -1
```

También se puede encontrar la última ocurrencia con `lastIndexOf()`:
```javascript
let animals = ['cat', 'dog', 'fish', 'rabbit', 'hamster', 'dog'];
let lastDog = animals.lastIndexOf('dog');
console.log(lastDog); // 5
```
### Sorting
`sort()` este método ordena los números de menor a mayor y las cadenas de la A-Z:
```javascript
let names = ["James", "Alicia", "Fatiha", "María", "Bert"];
names.sort();
console.log(names); // ["Alicia", "Bert", "Fatiha", "James", "María"]
```
### Reversing
`reverse()` este método revuelca la lista el último va al principio y así sucesivamente:
```javascript
names.reverse(); // ["María", "James", "Fatiha", "Bert", "Alicia"]
```
## Arrays multidimensionales
Un array puede contener otros arrays, también conocido como lista de listas:
```javascript
let arrOfArrays = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];
console.log(arrOfArrays); // [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
let value1 = arrOfArrays[0][0]; // 1
let value2 = arrOfArrays[2][2]; // 9

let arrOfArraysOfArrays = [arrOfArrays, arrOfArrays, arrOfArrays];
console.log(arrOfArraysOfArrays);
// [[[1, 2, 3], [4, 5, 6], [7, 8, 9]], [[1, 2, 3], [4, 5, 6], [7, 8, 9]], [[1, 2, 3], [4, 5, 6], [7, 8, 9]]]
let middleValue = arrOfArraysOfArrays[1][1][1]; // 5
```
## Objetos en JavaScript
- Los objetos describen cosas reales, también conceptos abstractos. **Los arrays son objetos con propiedades indexadas.**
- **Los objetos que estudiaremos tiene propiedades con nombre**, en lugar de un número de índice generado automáticamente, le asignaremos un nombre descriptivo personalizado.

Los objetos en JavaScript (también en otros lenguajes) tienen propiedades y pueden realizar acciones:
```javascript
let dog = {
    dogName: "JavaScript",
    weight: 2.4,
    color: "brown",
    breed: "chihuahua",
    age: 3,
    burglarBiter: true
};
```
Creamos un objeto perro con sus propiedades (características).

Para saber si algo debería ser una propiedad, puedes usar la siguiente oración: `object-name tiene un(a) nombre-de-propiedad`.

Hay dos formas de acceder una propiedad del objeto:
```javascript
let dogColor1 = dog["color"];
let dogColor2 = dog.color;
```
### Actualización de objetos
Podemos cambiar el valor de las propiedades de los objetos, pero tenemos dos opciones:
```javascript
dog["color"] = "blue";
dog.weight = 2.3;
```
Es útil sabe saber que si cambiamos el tipo de dato de una propiedad, no pasa nada.
## Trabajar con objetos y arrays
### Objetos en objetos
```javascript
let company = {
    companyName: "Health Candy",
    activity: "food manufacturing",
    address: {
        street: "123 Candy Lane",
        number: "123",
        zipCode: "12345",
        city: "Sweetville",
        state: "Florida"
    },
    yearOfEstablishment: 2005,
};

company.address.zipCode = "33117";
company["address"]["number"] = "100";
```
### Arrays en objetos
```javascript
let company = {
    companyName: "Health Candy",
    activities: ["food manufacturing", "improving kids' health", "manufacturing toys"],
    address: {
        street: "123 Candy Lane",
        number: "123",
        zipCode: "12345",
        city: "Sweetville",
        state: "Florida"
    },
    yearOfEstablishment: 2005,
};

let activity = company.activities[1]; // "improving kids' health"
```
### Objetos en arrays
Digamos que en nuestra empresa tengamos una lista de direcciones, podemos lograr creando un array de objetos de dirección:
```javascript
let addresses = [{
    street: "2nd street",
    number: "123",
    zipcode: "33116",
    city: "Miami",
    state: "Florida"
},
{
    street: "1st West Avenue",
    number: "5",
    zipcode: "75001",
    city: "Addison",
    state: "Texas"
}];

let streetName = addresses[0].street; // "2nd street"
```
