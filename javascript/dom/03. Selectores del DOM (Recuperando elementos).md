- Para manipular elementos, primero debemos encontrarlos en el árbol del DOM. 
- JavaScript moderno prioriza la flexibilidad usando la sintaxis de CSS.
## Selectores de elemento único
Estos métodos devuelven el primer elemento que coincida con la búsqueda. Si no encuentra nada, devuelve `null`.
1. `document.getElementById("id")
	- Método: Tradicional y muy rápido.
	- Sintaxis: Solo acepta el nombre del ID (sin `#`).
	- Uso: Ideal cuando el ID es conocido y único.
2. `document.querySelector("selectorCSS")`
	- Método: Moderno y estándar.
	- Sintaxis: Acepta cualquier selector CSS válido: (`.clase`, `#id`, `tag`, `padre hijo`).
	- Comportamiento: Devuelve solo la primera coincidencia que encuentra en el documento (lectura de arriba a abajo).
## Selectores de elementos múltiples
Devuelven una colección de nodos.
1. `document.querySelectorAll("selectorCSS")`
	- Retorno: Devuelve una `NodeList` (lista de nodos).
	- Importante: Una `NodeList` **No es un Array** de JavaScript, aunque se parece mucho. Es una colección estática.
	- Capacidades:
		- Tiene propiedad `.length`.
		- Se puede acceder por índice `[0]`.
		- En navegadores modernos soporta `.forEach()`.
		- No soporta `map`, `filter`, `reduce` directamente.
