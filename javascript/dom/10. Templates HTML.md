La etiqueta `<template>` es un mecanismo para mantener contenido HTML inerte (no se renderiza, no carga imágenes, no ejecuta scripts) hasta que decidimos usarlo mediante JavaScript. Es el "molde" ideal para crear elementos dinámicos repetitivos.
## El flujo de trabajo correcto
1. Capturar el contenido: Debes acceder a la propiedad `.content` del template.
2. Clonar: Creas una copia del molde.
3. Modificar: Rellenas la copia con tus datos.
4. Insertar: Agregas la copia al DOM (o a un fragmento).
