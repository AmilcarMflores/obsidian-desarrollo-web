En el navegador, existe una jerarquía estricta:
1. Window: Es el objeto global del navegador (BOM). Controla la ventana, el historial, la barra de direcciones, etc.
2. Document: Es una propiedad de `window`(`window.document`). Representa el DOM (el contenido HTML renderizado).

Como `window` es el objeto global, no es necesario escribirlo explícitamente. `window.document` es igual a `document`.
### Propiedades de acceso directo (estructurales)
El DOM ofrece accesos directos (atajos) a las partes principales de la estructura HTML:
- **`document.documentElement`**: Retorna el elemento raíz `<html>`.
    
- **`document.head`**: Retorna el elemento `<head>`.
    
- **`document.body`**: Retorna el elemento `<body>`.
    
- **`document.doctype`**: Retorna la definición del tipo de documento (ej: `<!DOCTYPE html>`).
### Colecciones de elementos (HTMLCollections)
Existen propiedades que devuelven listas de elementos específicos. Son útiles para diagnósticos rápidos, aunque en desarrollo moderno solemos usar selectores más específicos (`querySelector`).

- **`document.links`**: Todos los enlaces `<a>` con atributo `href`.
    
- **`document.scripts`**: Todos los archivos `<script>`.
    
- **`document.styleSheets`**: Todas las hojas de estilo CSS enlazadas.
    
- **`document.forms`**: Todos los formularios.
    
- **`document.images`**: Todas las etiquetas `<img>`.
