Para modificar la apariencia visual de forma dinámica, la mejor práctica no es alterar los estilos línea por línea (`.style`), sino añadir o quitar **Clases CSS** que ya tenemos definidos en la hoja de estilos.
## Propiedades de acceso
1. `className` (String)
	- Devuelve una cadena de texto simple con todas las clases.
	- Es difícil de gestionar si hay muchas clases. Si asignas un valor (`el.className = "rojo"`), borras todas las clases anteriores.
	- Uso actual: Poco frecuente, principalmente para lectura o limpieza total.
2. `classList` (DOMTokenList)
	- Retorno: Un objeto especial similar a un Array (DOMTokenList).
	- Ventaja: Ofrece métodos específicos para añadir, quitar o alternar clases sin afectar a las demás. Es el **estándar moderno**.
3. Métodos de `classList`
	- `contains("clase")`: Devuelve `true` si el elemento tiene la clase. Útil para condicionales.
	- `add("clase1", "clase2")`: Añade una o varias clases. Si ya existen, las ignora (no duplica).
	- `remove("clase1", "clase2")`: Elimina una o varias clases.
	- `toggle("clase")`: Interruptor. Si la tiene, la quita; si no la tiene, la pone.
	- `replace("vieja", "nueva")`: Reemplaza una clase por otra.
