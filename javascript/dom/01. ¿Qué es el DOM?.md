- El DOM es una interfaz de Programación de Aplicaciones (API) para documentos web (HTML y XML). 
- Cuando el navegador carga una página web, "parsea" (analiza) el código HTML  y genera una representación estructural en memoria conocida como Árbol del DOM (DOM Tree).

**Acrónimo:**
- Document: Hace referencia a la página web completa cargada en el navegador.
- Object: Cada componente del HTML (etiquetas, atributos, texto) se convierte en un objeto de JavaScript (nodos) con propiedades y métodos propios.
- Model: Es la representación jerárquica en forma de árbol que organiza estos objetos, definiendo las relaciones padre-hijo entre ellos.

**Por eso, el DOM actúa como un puente que conecta el lenguaje de programación (JavaScript) con el documento web, permitiendo:**
- **Leer y modificar la estructura (HTML).**
- **Cambiar el estilo visual (CSS).**
- **Reaccionar a eventos del usuario.**
