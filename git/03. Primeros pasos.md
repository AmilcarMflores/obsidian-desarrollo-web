#### 1. Inicializando un repositorio en un directorio existente
Debes ir al directorio del proyecto y usar el siguiente comando:
```bash
git init
```
Esto crea una subdirectorio llamado `.git` el cual contiene todos los archivos los archivos necesarios del repositorio.

Los archivos ocultos, podemos verlo con el comando:
```bash
ls -a
```
#### 2. Revisando el estado de tus archivos
Para determinar qué archivos están en qué estado se usa el comando `git status`. Este comando te indica en cuál rama estás y te informa del estado de tus archivos.
#### 3. Rastrear archivos nuevos
Debes usar el comando `git add <nombre_del_archivo>` o también:
```bash
# Añade todos los archivos nuevos
git add .
```
`git add` es un comando para **añadir este contenido a la próxima confirmación**.
#### 4. Confirmar cambios
La manera más sencilla es con `git commit -m <mensaje>`, **la confirmación guarda una instantánea de tu área de preparación**.
#### Saltar el área de preparación
Añadiendo la opción `a` al comando `git commit` haces que Git prepare y confirme los archivos que Git ya conoce (los que ya estaban bajo seguimiento y fueron modificados o eliminados):
```bash
git commit -a -m 'added new benchmarks'
```
----
#### Notas
- Cada archivo de un repositorio puede tener dos estados:
	- **Rastreados**: son aquellos archivos que estaban en la última instantánea del proyecto; pueden ser archivos sin modificar, modificados o preparados.
	- **Sin rastrear**: son todos los demás, cualquier otro archivo en un directorio de trabajo que no estaba en su última instantánea y que no está en el área de preparación.
- Si ejecutas `git status -s` o `git status --short`, obtendrás una salida simplificada:
```bash
git status -s
 M README
MM Rakefile
A  lib/git.rb
M  lib/simplegit.rb
?? LICENSE.txt
```
donde:
	- Los archivos nuevos que no están rastreados tienen un `??` a su lado.
	- Los archivos que están preparados tienen una `A`
	- Los modificados una `M`
	- La columna de la izquierda indica el estado preparado.
	- La columna de la derecha indica el estado sin preparar.
