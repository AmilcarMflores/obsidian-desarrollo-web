Un **tema de color** es un conjunto de colores armonizados que definen la identidad visual de un sitio.
## Dominando la teoría del color con HSL en CSS
El modelo HSL (Hue, Saturation, Lightness) ofrece una forma más intuitiva de trabajar con colores en CSS:
```css
color: hsl(120, 100%, 50%); /* Verde puro */
```

![[Pasted image 20250831210323.png|centrear-imagen]]

1. **Matiz (Hue)**:
    - Rango: 0° a 360°
    - 0° = Rojo, 120° = Verde, 240° = Azul
    - Representa el color base en la rueda cromática
2. **Saturación (Saturation)**:
    - Rango: 0% a 100%
    - 0% = Escala de grises, 100% = Color más intenso
    - Controla la pureza del color
3. **Luminosidad (Lightness)**:
    - Rango: 0% a 100%
    - 0% = Negro, 50% = Color puro, 100% = Blanco
    - Determina la claridad u oscuridad
### HSL vs Hexadecimal
| **Característica**    | **Hexadecimal**    | **HSL**             |
| --------------------- | ------------------ | ------------------- |
| Intuición             | Baja (abstracto)   | Alta (visual)       |
| Modificar luminosidad | Complejo           | Muy fácil           |
| Crear variaciones     | Requiere cálculo   | Con ajustes simples |
| Compatibilidad        | Universal          | Universal           |
| Accesibilidad         | Difícil de ajustar | Fácil de ajustar    |
Para obtener colores opuestos para el tema oscuro, se obtienen al agregar 180 grados al valor de matiz de cada color original. Si el valor excede a 360 grados, vuelve a empezar en 0 (rueda de colores).
## Propiedad color-scheme (tema oscuro)
La propiedad color-scheme y la media query prefers-color-scheme son herramientas esenciales para crear experiencias web que respeten las preferencias del usuario.
### Propiedad color-scheme
```css
:root {
  color-scheme: light dark; /* Soporta ambos temas */
}
```

**Valores y efectos:**
- **`light`**: Usa esquema claro para componentes del sistema
- **`dark`**: Usa esquema oscuro para componentes del sistema
- **`only light`/`only dark`**: Bloquea el esquema (evita Auto Dark Theme)
- **`normal`**: Comportamiento por defecto (no aplica ajustes)
### Media Query prefers-color-scheme
Detecta la preferencia de tema del sistema operativo:
```css
@media (prefers-color-scheme: light) {
  /* Estilos para tema claro */
}

@media (prefers-color-scheme: dark) {
  /* Estilos para tema oscuro */
}
```
## Explorando la función light-dark() en CSS
Simplifica la creación de temas claros y oscuros.
### Sintaxis y Comportamiento
```css
.elemento {
  propiedad: light-dark(
    valor-para-tema-claro,
    valor-para-tema-oscuro
  );
}
```

**Mecanismo de funcionamiento:**
1. Si el usuario prefiere tema claro o no tiene preferencia → Usa primer valor
2. Si el usuario prefiere tema oscuro → Usa segundo valor
3. **Requiere** **`color-scheme: light dark`** en el elemento padre.

Ejemplo básico
```css
:root {
  color-scheme: light dark;
}

body {
  background-color: light-dark(#fcf6ed, #121212);
  color: light-dark(#061b1a, #e0e0e0);
}

button {
  background: light-dark(#0c457d, #6bd2db);
  color: light-dark(white, #121212);
}
```
