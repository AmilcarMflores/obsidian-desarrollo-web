## Formularios en HTML
Los formularios permiten a los usuarios **interactuar con un sitio web** enviando información.
### form
- Contiene todos los elementos del formulario.
- Atributos principales:
    - **action:** URL a la que se envían los datos. 
    - **method:** método de envío (`get` para datos no sensibles, `post` para datos seguros o grandes cantidades).
```html
<form action="url_to_send_form_data" method="post">
  ...
</form>
```
### input
- Crea **campos de texto, casillas de verificación y botones de opción**.
- Atributos esenciales:
    - **type:** define el tipo de entrada.
    - **name:** nombre único para enviar los datos en pares clave-valor.
```html
<input type="text" name="firstname">
<input type="text" name="lastname">
```

- **maxlength:** limita la cantidad de caracteres.
```html
<input type="text" name="username" maxlength="20">
```
#### Tipos de inputs
- **Checkbox:** permite seleccionar múltiples opciones.
```html
<input type="checkbox" name="color1" value="red">Red
```

- **Radio:** permite seleccionar solo una opción. 
```html
<input type="radio" name="color" value="red">Red
```
### label
- Asocia un texto a un elemento de formulario.
- Permite seleccionar el input haciendo clic en el texto.
```html
<label for="first_name">First name:</label> <input type="text" id="first_name" name="firstname">
```
### textarea
- Permite capturar **múltiples líneas de texto**.
- Se puede ajustar con `rows` y `cols`.
```html
<textarea id="message" rows="5" cols="20"></textarea>
```
### fieldset
Agrupa elementos relacionados dentro del formulario.
### select y option
- Crea **listas desplegables** para seleccionar una opción.
```html
<select id="countries">
  <option value="england">England</option>
  <option value="scotland">Scotland</option>
  <option value="ireland">Ireland</option>
</select>
```
### button
- Atributo **type** determina su comportamiento:
    - **button:** sin acción predeterminada.
    - **submit:** envía el formulario.
    - **reset:** restablece los valores del formulario.
```html
<button type="submit">Send</button>     
<button type="reset">Clear</button>     
<button type="button">Click me</button>
```
## Estilando formularios en HTML y CSS
### Labels, Textboxes y Textareas
Ejemplo de estilos básicos aplicados a campos de texto y áreas de texto:
```html
<style>
  :root {
    --border-color: #666;
  }
  * {
    font-family: Arial, sans-serif;
  }
  label {
    font-size: 20px;
  }
  div {
    margin-bottom: 30px;
  }
  input, textarea {
    border: 0;
    border-bottom: 1px solid var(--border-color);
    padding: 10px 0;
    width: 200px;
  }
</style>

<form action="url_to_send_form_data" method="post">
  <div>
    <label for="first_name">First name:</label><br />
    <input type="text" name="firstname" id="first_name" placeholder="Your first name" />
  </div>

  <div>
    <label for="last_name">Last name:</label><br />
    <input type="text" name="lastname" id="last_name" placeholder="Your last name" />
  </div>

  <div>
    <label for="message">Message:</label><br />
    <textarea id="message" rows="5" cols="20" placeholder="Your message"></textarea>
  </div>
</form>
```

### Botones (button)
Ejemplo de estilos con efectos **hover** y **active**:
```html
<style>
  :root {
    --bg-color: #999;
    --bg-active-color: #888;
    --text-color: #fff;
  }
  button {
    background: var(--bg-color);
    border: 0;
    color: var(--text-color);
    cursor: pointer;
    font-size: 12px;
    height: 50px;
    width: 200px;
    text-transform: uppercase;
  }
  button:hover {
    background: var(--bg-active-color);
  }
  button:active {
    background: var(--bg-color);
  }
</style>

<button type="submit">Submit</button>
```

### Listas Desplegables (select)
Se pueden personalizar con CSS para que se vean más modernos:
```html
<style>
  :root {
    --border-color: #666;
  }
  select {
    background: transparent;
    border: 0;
    border-bottom: 1px solid var(--border-color);
    color: var(--border-color);
    padding: 10px 0;
    width: 200px;
    -webkit-appearance: none; /* quita el estilo por defecto */
  }
  .select-wrapper {
    position: relative;
    width: 200px;
  }
  .select-wrapper:after {
    content: '< >';
    color: var(--border-color);
    font-size: 14px;
    top: 8px;
    right: 0;
    transform: rotate(90deg);
    position: absolute;
  }
</style>

<div class="select-wrapper">
  <select id="countries">
    <option value="england">England</option>
    <option value="scotland">Scotland</option>
    <option value="ireland">Ireland</option>
    <option value="wales">Wales</option>
  </select>
</div>
```

## Validación con required
Obliga a rellenar un campo antes de enviar:
```html
<input type="text" name="username" required>
```
