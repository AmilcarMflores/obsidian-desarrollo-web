## HTML (HyperText Markup Language)
- Es un lenguaje que nos permite ==definir la estructura== de una página web.
- Se compone de etiquetas y atributos:

### Estructura
- Lo primero que se escribe es `<!DOCTYPE html>`, que indica al navegador que es un documento HTML5.
- Luego, la etiqueta `<html>` y dentro sus dos hijos: `<head>` y `<body>`.
## CSS (Cascading Style Sheets)
- Es un lenguaje que se usa para crear el ==diseño visual== de una página web.

![[Pasted image 20251021193757.png|centrear-imagen]]

- Una declaración CSS consta de una propiedad y un valor: `color: red;`. Esta declaración debe ir dentro de un selector.
- Para añadir estilos a un documento html, hay varias maneras, pero usaremos un archivo  *.css* .
### Selector universal (\*)
En CSS, para seleccionar todos los elementos de un documento HTML, usa este selector **\***.
### Selectores de atributos
- `[attribute]` Selecciona todos los elementos que contienen un atributo específico, sin importar su valor.
- `[attibute=value]` Selecciona todos los elementos que tienen un atributo con un valor exacto.
### Pseudo-clases
- Las **pseudo-clases** permiten aplicar estilos en función del **estado** o **posición** de un elemento en el documento.  
- Se escriben con **dos puntos (`:`)** seguidos de una palabra clave.

**Ejemplo con enlaces `<a>`:**
	- **`:link`** → Enlace no visitado.
	- **`:visited`** → Enlace ya visitado.
	- **`:hover`** → Cuando el cursor pasa por encima.
	- **`:active`** → Mientras se está haciendo clic en él.

Regla de orden (nemotecnia “LoVe-HAte”):   `link → visited → hover → active`  .Esto evita conflictos entre estilos.

**Otras pseudo-clases importantes:**
	- **`:focus`** → Se aplica cuando un elemento (ej. un `<input>`) está activo tras hacer clic o al navegar con teclado.
	- **`:first-child`** → Selecciona el primer hijo de un elemento padre.
	- **`:last-child`** → Selecciona el último hijo de un elemento padre.
	- **`:nth-child(n)`** → Selecciona el hijo _n_ de un padre (puede ser un número fijo o un patrón como `2n` para pares, `2n+1` para impares).
### Pseudo-elementos
- Los **pseudo-elementos** permiten dar estilo o insertar contenido en **partes específicas de un elemento**, sin necesidad de modificar el HTML.  
- Se escriben con **doble dos puntos (`::`)**

**Pseudo-elementos más comunes:**
	- **`::before`** →  Inserta contenido **antes** del contenido real de un elemento.
	- **`::after`** →  Inserta contenido **después** del contenido real de un elemento.
	- **`::selection`** →  Permite cambiar el estilo del texto cuando el usuario lo selecciona.
	- **`::first-line`** → Estiliza solo la **primera línea** de un párrafo.
	- **`::first-letter`** → Estiliza solo la **primera letra** de un párrafo (útil para efectos de capitulares).

## Variables en CSS
- Las **variables en CSS** (también llamadas _Custom Properties_) permiten **almacenar valores reutilizables** como colores, tamaños o espacios.  
- Son muy útiles para mantener la **consistencia** en el diseño y facilitan la **modificación global** de estilos en proyectos grandes.
### Sintaxis
- Se definen con el prefijo `--` dentro de un selector (comúnmente en `:root` para que sean globales).
- Se llaman con la función `var()`.
- Pueden tener un **valor de respaldo** en caso de que la variable no esté definida.

**Ejemplo práctico:**
```css
/* Definir variables globales */
:root {
  --color-primario: #FC9C9C;
  --color-secundario: #FF5733;
  --tamaño-texto: 18px;
  --espaciado: 10px;
}

body {
  background-color: var(--color-primario);
  font-size: var(--tamaño-texto);
}

p {
  color: var(--color-secundario, #FF0000); /* fallback: rojo */
  padding: var(--espaciado);
}

.caja {
  background-color: white;
  padding: var(--espaciado);
  border: 2px solid var(--color-secundario);
}
```
